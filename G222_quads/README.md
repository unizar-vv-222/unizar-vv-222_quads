# G222 Quads - Android App

Aplicación Android desarrollada para la gestión de quads y reservas.

La aplicación permite:

* Gestión de quads
* Gestión de reservas
* Asociación de quads a reservas
* Envío de información de reservas
* Ejecución de pruebas sobre los distintos módulos

---

# Estructura del proyecto

El proyecto sigue una arquitectura basada en:

* Activities y Fragments para la interfaz gráfica
* ViewModels para la gestión del estado
* Repositories para el acceso a datos
* Base de datos Room/SQLite
* Recursos XML para layouts y estilos

Además, el proyecto incluye:

* Pruebas unitarias
* Pruebas instrumentadas
* Recursos Material Design

---

# Requisitos

* Android Studio
* SDK Android compatible
* Gradle

---

# Ejecución

1. Abrir la carpeta `G222_quads` con Android Studio.
2. Sincronizar dependencias Gradle.
3. Ejecutar la aplicación en un emulador o dispositivo Android.
